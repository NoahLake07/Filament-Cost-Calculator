# Filament Cost Calculator
A simple app designed to calculate the price per meter and length of a 3D filament spool.

Here's what you'll need to know about your filament:
1. Material (PLA, ABS, PETG, TPU, etc.)
2. Price ($)
3. Weight (kg)
